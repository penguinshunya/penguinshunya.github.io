setのループで要素を削除する

setのループを範囲for文で書き、その中でsetの要素を削除すると、よくわからない挙動を見せます。以下のように書くことで正しく動作します。 
set<int> s = {1, 2, 3};

auto it = s.begin();
while (it != s.end()) {
  if (*it >= 3) break;
  s.erase(it++);
}

for (auto v : s) {
  cout << v << '\n'; //=> 3
}
